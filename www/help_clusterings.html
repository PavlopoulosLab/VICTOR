This tab generates all clustering comparison data. The user can select among 10 clustering comparison metrics. This tab consists of two sub-tabs: (i) Explore Metric result and (ii) Maximum Cluster Match.

Explore Metric Results

After generating the clustering comparison results, the values are annotated in a color-gradient table. Column names describe the pairwise comparison of the clustering files (A vs B, A vs C, etc.) and row names the user-selected metrics. Color values range from light yellow (low metric value) to red (high metric value).

Maximum Cluster Match

The user can select two of the uploaded clustering files and view their maximum cluster overlaps based on the Maximum Match Measure algorithm. The result table contains the matched clusters from the two clustering files and their respective object overlap in descending order.
