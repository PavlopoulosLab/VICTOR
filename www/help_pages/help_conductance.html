This tab consists of two sub tabs: (i) Network and (ii) Conductance Histogram.

Conductance
In this tab, the user can see how well a cluster describes a given network. Users must upload a network file in edge-list format and associate it with a clustering file of interest.

Network file format
The network file is a two- or three-column (unweighted or weighted respectively) text file in which the two first columns describe the source and target nodes and the optional third column their in-between edge weights.

Network
In this sub-tab, the uploaded network is visualized. After the user chooses a clustering file, network nodes are colored accordingly. Notably, all objects contained in the clustering file must be present in the network.

Conductance Histogram
In this sub-tab, the histogram of the conductance of the various clusters is visualized. The horizontal axis of the histogram shows the conductance scores and the vertical axis shows the number of clusters corresponding to each score. The more the plotted bars tilt towards 0, the better the clusters' conductance.
