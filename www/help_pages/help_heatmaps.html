This tab visualizes the comparison between the clustering files as a hierarchical heatmap.

Hierarchical Heatmap
A hierarchical heatmap clusters objects in both x and y axes, according to their similarity. The color depth of a cell is proportional to the pairwise similarity value. In VICTOR, the hierarchical heatmap visualization is a n x n table, where n represents the number of the user-selected clustering files. A dendrogram is appended on both axes illustrating the cluster arrangement of the in-between pairwise similarity. Color values range from light yellow (low similarity) to deep red (high similarity).
